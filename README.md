<h1>Inky Binky</h1>

Step into a world of raw sketches and rapid-fire combat. In Inky Binky, you control a uniquely designed character fighting to stay alive against encroaching enemies on an infinite, hand-drawn battlefield.

Designed with a clean, sketchy aesthetic, the game focuses on pure movement and survival. Whether you’re playing on a PC with a keyboard or on the go with your mobile device, the experience is optimized for smooth, responsive action.
Technical Specs (Important for itch.io)

    Platforms: Windows, Android, Web (HTML5).
    Controls: WASD/Arrows for Movement (PC), On-screen Touch Controls (Mobile).
    Engine: Powered by Godot 4.
